5. 선택문(삼항연산자/switch)

switch 문

switch( 정수형 변수 ) {
  case 첫번째 정수값:
    변수값이 첫번째 정수값이면 실행하는 명령문
    break;
  case 두번째 정수값:
    변수값이 두번째 정수값이면 실행하는 명령문
    break;
  ...
  default:
    변수값이 앞에서 제시된 정수값이 아니면 실행하는 명령문
}

switch   정수형 변수가 특정 정수값이면 해당 명령문을 실행하는 선택문

case   switch 문에서 정수형 변수를 특정 정수값과 비교하는 명령문

:   case문에서 정수형 변수와 특정 정수값을 비교할 때 사용되는 기호

break   실행흐름을 중단하고 switch 문을 빠져나오는 명령문

default   switch 문에서 정수형 변수의 제시된 정수값이 없을 때를 표현

import java.util.Scanner;

public class Main {

        case 9:

            break;
        case 8:

삼항 연산자

조건식 ? 조건식이 참일 때 명령어 : 조건식이 거짓일 때 명령어

?   삼항연산자에서 조건식이 참인지 거짓인지 확인하는 기호

:   삼항연산자에서 조건식이 참과 거짓일 때 실행하는 명령문을 구분하는 기호

import java.util.Scanner;
import java.lang.Math;

public class Main {
  public static void main( String[] args ) {
    Scanner scan = new Scanner( System.in );
    int user = 0, coin = (int)( Math.random() * 2 );

    System.out.println( "동전 던지기 게임을 시작합니다." );
    System.out.print( "앞면(0), 뒤면(1) 중 하나를 선택하세요: " );
    user = scan.nextInt();

    System.out.println( ( coin == user ) ? "맞았습니다" : "틀렸습니다" );
  }
}

java.lang.Math   임의의 수를 생성하는 메서드를 제공해주는 클래스의 패키지

Math.random   0 ~ 1 사이의 실수 중 임의의 수 하나를 반환하는 메소드
